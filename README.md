# Практическая работа №1

## Тема

Разработка программного кода для устройств промышленного интернета вещей на формальном алгоритмическом языке.

## Автор

Коровин Данила Юрьевич, группа ВТ-1-24.

## Описание проекта

Проект представляет программу мониторинга температуры и влажности для IIoT-устройства. Программа имитирует работу датчика DHT22, управление вентиляцией через реле и отправку данных на MQTT-брокер.

## Функциональность

- опрос датчика температуры и влажности каждые 2 секунды;
- включение вентиляции при температуре выше 35°C;
- выключение вентиляции при температуре ниже 28°C;
- отправка данных на MQTT-брокер каждые 10 секунд;
- формирование сообщения в формате JSON;
- буферизация данных при потере MQTT-связи;
- вывод диагностических сообщений в терминал.

## Файлы проекта

- `main.py` - основной исходный код программы;
- `README.md` - описание проекта;
- `.gitignore` - список файлов, которые не нужно загружать в репозиторий.
